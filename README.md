Gráficos Mercado de Ações

Projeto desenvolvido na formação de Analista de Dados da EBAC.

Objetivo

Criar visualizações gráficas para facilitar a interpretação dos dados de operações do mercado de ações.

Gráficos Desenvolvidos

Estatística

* Gráfico de Colunas
* Estatísticas do preço das ações

Tendência Temporal

* Gráfico de Linha
* Evolução das operações ao longo de 2023

Compra e Venda

* Gráfico de Pizza
* Distribuição das operações

Cliente Compra e Venda

* Gráfico de Barras Horizontais
* Volume operado por cliente

Cliente Lucro e Prejuízo

* Gráfico de Área
* Resultado financeiro por ativo

Mercado de Ações

* Gráfico de Dispersão
* Relação entre preço e quantidade negociada

Ferramentas Utilizadas

* Microsoft Excel / Google Sheets
* Tabelas Dinâmicas
* Visualização de Dados
* GitHub
